# 🏰 Math Quest: سرزمین جادویی اعداد

یک بازی تحت وب فانتزی و آموزشی برای یادگیری ریاضیات (جمع، تفریق، ضرب و تقسیم) مخصوص کودکان.

## 🚀 ویژگی‌ها
- **۸ مرحله داستانی:** سفر از جنگل اعداد تا کهکشان بی‌پایان.
- **چالش‌های متنوع:** هر مرحله دارای هدف خاص و داستان منحصربه‌فرد است.
- **نقشه مراحل:** نمایش پیشرفت کاربر با قابلیت باز شدن قفل مراحل.
- **تالار قهرمانان:** سیستم ثبت رکوردهای برتر به صورت آفلاین.
- **طراحی مدرن:** استفاده از انیمیشن‌های جذاب و تم فانتزی.
- **کاملاً واکنش‌گرا:** قابل اجرا روی موبایل، تبلت و کامپیوتر.

## 🛠 تکنولوژی‌های استفاده شده
- **React** (Vite)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (برای انیمیشن‌ها)
- **Lucide React** (برای آیکون‌ها)

## 📦 نصب و اجرا
۱. ابتدا پروژ را کلون کنید یا فایل‌ها را دانلود کنید.
۲. با استفاده از دستور زیر کتابخانه‌ها را نصب کنید:
```bash
npm install
```
۳. برای اجرای پروژه در حالت توسعه:
```bash
npm run dev
```
۴. برای ساخت نسخه نهایی:
```bash
npm run build
```

ساخته شده با ❤️ برای آموزش بهتر کودکان.
